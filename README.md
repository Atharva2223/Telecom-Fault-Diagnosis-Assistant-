# Telecom Fault Diagnosis Assistant using LoRA and RAG

This project builds a telecom fault diagnosis assistant using telecom-style network incident logs, LoRA fine-tuning, and RAG-based troubleshooting.

## Features

- Telecom fault classification
- Root-cause detection
- LoRA/QLoRA fine-tuning
- RAG using LangChain and Pinecone/FAISS
- Evaluation using RAGAS and DeepEval
- Streamlit demo app

## Dataset

This project uses a telecom-style dataset derived from public system log datasets and KPI simulation.
